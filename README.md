# compare_competencies
Сравнение компетенций учебной программы и с job-сайтами
Ориентировочное древо проекта:
```plaintext
└── 📁 competency-comparison
    ├── 📁 data # Хранение всех данных (сырые, обработанные, ученики, результаты)
    │   ├── 📁 last_uploaded # Последняя загруженная версия компетенций учеников 
    │   │   └── 📄 competency_matrix.csv #Последний загруженный вариант компетенций
    │   ├── 📁 processed # Обработанные и нормализованные данные с hh.ru
    │   │   ├── 📄 competency_frequency.json # Словарь {навык: частота} для быстрого доступа
    │   │   ├── 📄 competency_mapping.json # Словарь {компетенция: ключевые слова}
    │   │   └── 📄 market_competencies.csv # Таблица с частотами навыков и категориями (для Pandas)
    │   ├── 📁 raw # Сырые данные, полученные с hh.ru (ответы API)
    │   │   ├── 📄 competency_matrix.csv #Отсюда идет подгруз в .../src/loaders_student/student_load.py
    │   │   └── 📄 hh_vacancies.json # Массив вакансий с полными данными (включая key_skills)
    │   ├── 📁 result # Результаты сравнения и рекомендации для разных учеников
    │   │   ├── 📁 base # Результаты для ученика "base"
    │   │   │   ├── 📄 comparison_report_base.json # Отчёт сравнения: общие навыки, дефициты, метрики
    │   │   │   └── 📄 recommendations_base.json # Персональные рекомендации по развитию
    │   │   ├── 📁 dc # Результаты для ученика "dc"
    │   │   │   ├── 📄 comparison_report_dc.json
    │   │   │   └── 📄 recommendations_dc.json
    │   │   └── 📁 top_dc # Результаты для ученика "top_dc"
    │   │       ├── 📄 comparison_report_top_dc.json
    │   │       └── 📄 recommendations_top_dc.json
    │   ├── 📄 it_skills.json #Вариация белого списка для навыков с хх (костыль, на доработке)
    │   └── 📁 students # JSON-файлы с компетенциями учеников (входные данные)
    │       ├── 📄 base_competency.json # Список навыков ученика "base"
    │       ├── 📄 dc_competency.json # Список навыков ученика "dc"
    │       ├── 📄 descriptiom_of_competency.txt
    │       └── 📄 top_dc_competency.json # Список навыков ученика "top_dc"
    │
    ├── 📁 notebooks # Jupyter ноутбуки для исследовательского анализа и прототипирования
    │   ├── 📄 01_hh_analysis.ipynb # Анализ рыночных данных с hh.ru (частоты, тренды, кластеризация)
    │   ├── 📄 02_competency_matching.ipynb # Сравнение профилей учеников с рынком, gap-анализ
    │   └── 📄 03_prediction_model.ipynb # Прототипирование рекомендаций и прогнозирование навыков
    │
    ├── 📁 src # Исходный код проекта (Python-модули)
    │   ├── 📁 analyzers # Модули сравнения и анализа дефицитов
    │   │   ├── 📄 comparator.py # Функции для сравнения навыков ученика с рыночными (пересечения, метрики)
    │   │   ├── 📄 gap_analyzer.py # Определение приоритетных дефицитов и группировка по категориям
    │   │   ├── 📄 init.py # Маркер пакета Python
    │   │   └── 📄 trends.py # Анализ временных трендов навыков (если есть исторические данные)
    │   │
    │   ├── 📁 loaders # Загрузка данных учеников из JSON
    │   │   ├── 📄 init.py
    │   │   └── 📄 student_loader.py # Функции для чтения и валидации JSON учеников (использует Pydantic)
    │   │
    │   ├── 📁 models # Pydantic-модели для данных (ученик, компетенция, вакансия)
    │   │   ├── 📄 competency.py # Модель рыночной компетенции (название, частота, категория)
    │   │   ├── 📄 init.py
    │   │   ├── 📄 student.py # Модель ученика (id, имя, список навыков, целевая профессия)
    │   │   └── 📄 vacancy.py # Модель вакансии (id, название, навыки, регион, зарплата)
    │   │
    │   ├── 📁 parsing # Модули для парсинга hh.ru (API и обработка)
    │   │   ├── 📄 hh_api.py # Функции для запросов к API hh.ru (получение вакансий, обработка пагинации)
    │   │   ├── 📄 init.py
    │   │   ├── 📄 utils.py # Вспомогательные функции для парсеров (очистка текста, ротация User-Agent)
    │   │   └── 📄 vacancy_parser.py # Извлечение навыков из вакансий, нормализация, подсчёт частот
    │   │
    │   ├── 📁 predictors # Модули прогнозирования и генерации рекомендаций
    │   │   ├── 📄 init.py
    │   │   ├── 📄 recommendation_engine.py # Подбор образовательных ресурсов под дефициты, построение траекторий
    │   │   └── 📄 skill_forecast.py # Модели предсказания востребованности навыков (если используется ML)
    │   │
    │   ├── 📁 visualization # Построение графиков и диаграмм
    │   │   ├── 📄 charts.py # Функции для визуализации (столбчатые, лепестковые диаграммы, heatmap)
    │   │   └── 📄 init.py
    │   │
    │   ├── 📄 config.py # Конфигурационные константы (пути к папкам, параметры API, пороги)
    │   ├── 📄 init.py # Маркер корневого пакета src
    │   └── 📄 utils.py # Общие утилиты (логирование, работа с JSON, создание папок)
    │
    ├── 📁 tests # Модульные тесты
    │   ├── 📄 init.py
    │   ├── 📄 test_comparator.py # Тесты для функций сравнения
    │   ├── 📄 test_loaders.py # Тесты загрузки JSON и валидации
    │   └── 📄 test_parsers.py # Тесты парсеров (с моками API)
    │
    ├── 📄 main.py # Точка входа: запуск полного пайплайна (парсинг -> сравнение -> рекомендации)
    ├── 📄 README.md # Описание проекта, установка, примеры использования
    └── 📄 requirements.txt # Список зависимостей Python (requests, pandas, pydantic, matplotlib и т.д.)
```
