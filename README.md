# compare_competencies
Сравнение компетенций учебной программы и с job-сайтами
Ориентировочное древо проекта:
└── 📁 competency-comparison      # Корневая папка проекта
    ├── 📁 data                   # Хранение всех данных (сырые, обработанные, ученики, результаты)
    │   ├── 📁 processed          # Очищенные и нормализованные данные после обработки
    │   │   ├── 📄 competency_frequency.json    # Частотный словарь навыков с рынка
    │   │   └── 📄 market_competencies.csv      # Таблица рыночных компетенций для анализа
    │   ├── 📁 raw                 # Сырые данные, полученные напрямую с hh.ru
    │   │   └── 📄 hh_vacancies.json            # Ответы API hh.ru (список вакансий)
    │   ├── 📁 results              # Результаты сравнения и прогнозы для учеников
    │   │   ├── 📄 comparison_report_student_1.json  # Отчёт о сравнении для ученика 1
    │   │   └── 📄 recommendations_student_1.json    # Рекомендации для ученика 1
    │   └── 📁 students             # Исходные JSON-файлы с компетенциями учеников
    │       ├── 📄 student_1.json                   # Данные ученика 1
    │       └── 📄 student_2.json                   # Данные ученика 2
    ├── 📁 notebooks                # Jupyter ноутбуки для исследований и прототипирования
    │   ├── 📄 01_hh_analysis.ipynb               # Анализ данных с hh.ru
    │   ├── 📄 02_competency_matching.ipynb       # Сравнение компетенций учеников с рынком
    │   └── 📄 03_prediction_model.ipynb          # Прогнозирование и генерация рекомендаций
    ├── 📁 src                      # Исходный код проекта
    │   ├── 📁 analyzers            # Модули анализа и сравнения компетенций
    │   │   ├── 📄 comparator.py                 # Сравнение профиля ученика с рыночными данными
    │   │   ├── 📄 gap_analyzer.py                # Анализ дефицитов (gap-анализ)
    │   │   ├── 📄 init.py                         # Маркер пакета Python
    │   │   └── 📄 trends.py                       # Анализ трендов востребованности навыков
    │   ├── 📁 loaders               # Загрузка данных из внешних источников (JSON учеников)
    │   │   ├── 📄 init.py                         # Маркер пакета
    │   │   └── 📄 student_loader.py               # Чтение и валидация JSON учеников
    │   ├── 📁 models                # Модели данных (Pydantic/dataclasses)
    │   │   ├── 📄 competency.py                   # Модель рыночной компетенции
    │   │   ├── 📄 init.py                         # Маркер пакета
    │   │   ├── 📄 student.py                       # Модель данных ученика
    │   │   └── 📄 vacancy.py                       # Модель вакансии
    │   ├── 📁 parsers               # Парсинг данных с hh.ru
    │   │   ├── 📄 hh_api.py                        # Работа с API HeadHunter
    │   │   ├── 📄 init.py                          # Маркер пакета
    │   │   ├── 📄 utils.py                         # Вспомогательные функции для парсеров
    │   │   └── 📄 vacancy_parser.py                 # Извлечение навыков из вакансий
    │   ├── 📁 predictors            # Модули прогнозирования и рекомендаций
    │   │   ├── 📄 init.py                          # Маркер пакета
    │   │   ├── 📄 recommendation_engine.py          # Генерация рекомендаций по развитию
    │   │   └── 📄 skill_forecast.py                  # Прогноз востребованности навыков
    │   ├── 📁 visualization         # Визуализация данных и результатов
    │   │   ├── 📄 charts.py                         # Построение графиков (Matplotlib/Plotly)
    │   │   └── 📄 init.py                           # Маркер пакета
    │   ├── 📄 config.py              # Конфигурация проекта (пути, параметры API, пороги)
    │   ├── 📄 init.py                # Маркер корневого пакета src
    │   └── 📄 utils.py               # Общие утилиты (логирование, работа с файлами)
    ├── 📁 tests                      # Модульные тесты
    │   ├── 📄 init.py                 # Маркер пакета тестов
    │   ├── 📄 test_comparator.py      # Тесты для модуля comparator
    │   ├── 📄 test_loaders.py         # Тесты для загрузчиков
    │   └── 📄 test_parsers.py         # Тесты для парсеров
    ├── 📄 main.py                     # Точка входа для запуска пайплайна
    ├── 📄 README.md                   # Описание проекта, инструкции по установке и запуску
    └── 📄 requirements.txt            # Зависимости Python
