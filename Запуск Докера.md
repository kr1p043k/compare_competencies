## 1. Системные требования

- Windows 10/11 Pro
- 8 GB RAM (рекомендуется 16 GB)
- 20 GB свободного места
- Docker Desktop установлен и запущен

## 2. Подготовка проекта

Открыть PowerShell в папке проекта и выполнить:

```powershell
New-Item -ItemType Directory -Force -Path "data\cache", "data\processed", "data\result", "data\history", "logs"
New-Item -Path .env -ItemType File -Force
```

## 3. Запуск API

```powershell
docker compose up -d --build
```

```powershell
curl http://localhost:8000/api/health
```

Открыть браузер: http://localhost:8000/docs

## 4. Запуск n8n

```powershell
docker run -d --name n8n -p 5678:5678 -v n8n-data:/home/node/.n8n -e N8N_HOST=0.0.0.0 -e GENERIC_TIMEZONE=Europe/Moscow --restart unless-stopped n8nio/n8n:latest
```

```powershell
curl http://localhost:5678/healthz
```

Открыть браузер: http://localhost:5678

## 5. Запуск DeepSeek

```powershell
docker run -d --name ollama-deepseek -p 11434:11434 -v ollama-data:/root/.ollama --restart unless-stopped ollama/ollama:latest
```

```powershell
Start-Sleep -Seconds 10
docker exec -it ollama-deepseek ollama pull deepseek-r1:1.5b
```

```powershell
$body = @{model = "deepseek-r1:1.5b"; prompt = "Привет"; stream = $false} | ConvertTo-Json
Invoke-RestMethod -Uri "http://localhost:11434/api/generate" -Method POST -Body $body -ContentType "application/json"
```

## 6. Проверка всех контейнеров

```powershell
docker ps
```

## 7. Остановка всех сервисов

```powershell
docker stop compare-competencies-api n8n ollama-deepseek
```

## 8. Запуск после остановки

```powershell
docker start compare-competencies-api n8n ollama-deepseek
```

## 9. Просмотр логов

```powershell
docker logs compare-competencies-api --tail=50
docker logs n8n --tail=50
docker logs ollama-deepseek --tail=50
```

## 10. Полное удаление

```powershell
docker stop compare-competencies-api n8n ollama-deepseek
docker rm compare-competencies-api n8n ollama-deepseek
docker volume rm n8n-data ollama-data
docker system prune -a -f
```

## 11. Решение проблем

### Порт занят
```powershell
netstat -ano | findstr :8000
taskkill /PID <PID> /F
```

### Модель не найдена
```powershell
docker exec -it ollama-deepseek ollama pull deepseek-r1:1.5b
```

### API перезапускается
```powershell
docker logs compare-competencies-api --tail=50
Remove-Item "data\processed\*.json" -Force
docker restart compare-competencies-api
```
