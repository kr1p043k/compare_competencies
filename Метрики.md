## 1. Базовые технические метрики (ваше железо)

Эти метрики вы можете проверить прямо сейчас, потому что они зависят от вашего ноутбука / сервера.

| Метрика | Ваше значение (DeepSeek 1.5B) | Пояснение |
|--------|-------------------------------|-----------|
| **Размер модели на диске** | ~1.5 GB | Занимает в Docker volume `ollama-data` |
| **RAM при загрузке** | ~2.5–3.5 GB | Потребление оперативной памяти |
| **RAM при инференсе** | ~3–4.5 GB | Во время генерации ответа |
| **Время первого токена (TTFT)** | 0.3–1 сек | Как быстро модель начинает отвечать |
| **Скорость генерации** | 80–150 токенов/сек | На CPU. Это **очень быстро** |
| **Длина контекста** | 8192 токенов | Может «помнить» примерно 6000 русских слов |

> Если вы запустите DeepSeek 7B — цифры станут другими (см. таблицу ниже)

## 3. Метрики именно в вашей Docker-среде

Это то, что вы можете измерить сами прямо сейчас.

### Доступность и время ответа

```powershell
# Ваша текущая метрика: время ответа DeepSeek через Docker
Measure-Command {
    $body = @{model="deepseek-r1:1.5b"; prompt="Привет"; stream=$false} | ConvertTo-Json
    Invoke-RestMethod -Uri "http://localhost:11434/api/generate" -Method POST -Body $body -ContentType "application/json"
}
```

**Ожидаемый результат:**
- **Время ответа**: 0.5–2 секунды
- **P99 latency**: ≤ 3 секунды

### Доступность API

```powershell
# Проверка uptime через Docker
docker ps --filter "name=ollama-deepseek" --format "table {{.Names}}\t{{.Status}}\t{{.RunningFor}}"
```

**Цель:**
- **Uptime**: 99.9% (контейнер не падает сам)
- **Restart count**: `docker inspect ollama-deepseek | grep RestartCount`

### Скорость токенов

```powershell
# Замер скорости генерации
$longPrompt = "Напиши небольшой рассказ о программисте, который..." * 50

$body = @{
    model = "deepseek-r1:1.5b"
    prompt = $longPrompt
    stream = $false
} | ConvertTo-Json

$elapsed = Measure-Command {
    $response = Invoke-RestMethod -Uri "http://localhost:11434/api/generate" -Method POST -Body $body -ContentType "application/json"
}
$tokens = ($response.response).Split().Count
$speed = $tokens / $elapsed.TotalSeconds

Write-Host "Скорость: $([math]::Round($speed, 1)) токенов/сек"
Write-Host "Токенов: $tokens"
Write-Host "Секунд: $($elapsed.TotalSeconds)"
```

**Ожидаемые значения:**
- CPU (без GPU): **80–150 токенов/сек**
- GPU (если есть): **300–800 токенов/сек**

## 5. Как улучшить метрики

Если вам нужна **более умная модель**:

```powershell
# Скачать 7B вместо 1.5B
docker exec -it ollama-deepseek ollama pull deepseek-r1:7b

# Удалить старую модель (освободить место)
docker exec -it ollama-deepseek ollama rm deepseek-r1:1.5b
```

Если вам нужна **скорость** (оставить 1.5B):

```powershell
# Уже настроено. Оптимально для чата и простых задач.
```

Если вам нужна **стабильность метрик**:

```powershell
# Добавить ограничение ресурсов в docker run
docker update --memory="4g" --cpus="2" ollama-deepseek

# Проверить логи на ошибки
docker logs ollama-deepseek --tail=50
```
