# Исправления в src/predictors/ — оставшиеся файлы

## Найденные проблемы

### 1. ltr_recommendation_engine.py — КРИТИЧЕСКИЙ БАГ
**Строки 296, 305:** Переменная `ndcg` используется, но никогда не определена в `fit()`.

```python
# Строка 296:
"ndcg_at_10": round(ndcg, 4) if not np.isnan(ndcg) else 0.0,
# Строка 305:
"ndcg": round(ndcg, 4) if not np.isnan(ndcg) else 0.0,
```

Метод `fit()` вычисляет `r2`, `mae`, `spear`, но **не вычисляет `ndcg`**. `ndcg_score` импортируется (строка 16), но нигде не вызывается. Вызов `fit()` вызовет `NameError`.

**Исправление:** Добавить вычисление ndcg перед сохранением:
```python
try:
    ndcg = ndcg_score(y_test, pred_test, k=min(10, len(y_test)))
except Exception:
    ndcg = float("nan")
```

### 2. recommendation_engine.py — Мёртвый код и duck-typing
**Строка 47:** `self.gap_analyzer: GapAnalyzer | None = None` — объявлено, но `GapAnalyzer` нигде не импортируется и атрибут никогда не используется.

**Строка 178:** `if not hasattr(self, "profile_evaluator") or self.profile_evaluator is None:` — `hasattr` всегда True (атрибут создаётся в `__init__`). Проверка избыточна.

**Строки 265, 277, 452:** `hasattr(self.profile_evaluator, "domain_analyzer")`, `hasattr(self.profile_evaluator, "clusterer")` — duck-typing проверки. Если `profile_evaluator` всегда `ProfileEvaluator`, эти проверки избыточны. Но если он может быть `None`, достаточно `if self.profile_evaluator and ...`.

**Строка 501:** `_empty_recommendations()` возвращает `dict`, а не `RecommendationResult`. Метод не вызывается в production code.

**Исправления:**
- Удалить `self.gap_analyzer` (строка 47)
- Заменить `hasattr` проверки на `if self.profile_evaluator is not None`
- Удалить `_empty_recommendations()` или исправить возврат на `RecommendationResult`

### 3. ltr_recommendation_engine.py — Моду_level side effects
**Строка 83:** `_SYNTHETIC_DOMAIN_PROFILES: list[list[str]] = _load_profession_profiles()` — чтение JSON файлов при импорте модуля. Если файлы отсутствуют, вызывает warning при каждом импорте.

**Исправление:** Сделать lazy loading:
```python
_SYNTHETIC_DOMAIN_PROFILES: list[list[str]] | None = None

def _get_domain_profiles():
    global _SYNTHETIC_DOMAIN_PROFILES
    if _SYNTHETIC_DOMAIN_PROFILES is None:
        _SYNTHETIC_DOMAIN_PROFILES = _load_profession_profiles()
    return _SYNTHETIC_DOMAIN_PROFILES
```

### 4. factory.py — Forward reference annotations
**Строки 16, 53:** Return type annotations используют строковые литералы (`"CrossEncoderReranker"`, `"RecommendationEngine"`), которые не резолвятся в runtime. Не критично для работы, но нарушает type safety.

**Исправление:** Использовать `from __future__ import annotations` или moved imports.
