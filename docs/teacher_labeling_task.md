# Задача на разметку: валидация рекомендаций пайплайна

## Для чего

У нас есть пайплайн, который для каждого студента выдаёт список рекомендованных навыков. Нужно понять, насколько эти рекомендации правильные.

## Что нужно сделать

Для каждого студента (3 профиля: base, dc, top_dc) проставить:
- **да** — навык действительно нужен
- **нет** — навык не релевантен / слишком сложный

## Сколько времени

~10-15 минут на человека (60 оценок)

## Как это поможет

1. Precision@k — сколько из топ-5 рекомендаций верны
2. Recall — сколько нужных навыков пайплайн нашёл
3. Объективное сравнение версий пайплайна
