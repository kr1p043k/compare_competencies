# Competency Gap Analyzer — описание проекта

## Суть проекта

Система анализирует соответствие учебных компетенций студентов IT-специальностей реальным требованиям рынка труда. На основе вакансий с hh.ru определяет, каких навыков не хватает студенту для трудоустройства, и выдаёт персонализированные рекомендации по дообучению.

## Что делает

1. **Собирает вакансии** с hh.ru по IT-профессиям
2. **Извлекает навыки** из текста вакансий, нормализует (синонимы, fuzzy-матчинг)
3. **Оценивает важность** каждого навыка через гибрид BM25 + эмбеддинги (SentenceTransformer)
4. **Сравнивает** навыки студента с рыночными требованиями (gap-анализ)
5. **Ранжирует** недостающие навыки через ML (XGBoost + SHAP) с объяснениями
6. **Генерирует** персонализированные рекомендации: что учить, почему важно, сколько времени займёт
7. **Визуализирует** результаты: радарные диаграммы, тепловые карты, графики покрытия

## Технологии

| Компонент | Технология |
|-----------|------------|
| Язык | Python 3.12 |
| Веб-сервер | FastAPI (REST API) |
| Модели данных | Pydantic v2 |
| Эмбеддинги | SentenceTransformers (paraphrase-multilingual-mpnet-base-v2) |
| BM25 | rank_bm25 (Okapi) |
| ML-ранжирование | XGBoost + SHAP |
| Кластеризация | scikit-learn (KMeans, HDBSCAN) |
| Нормализация текста | pymorphy3, rapidfuzz |
| Логирование | structlog |
| Визуализация | matplotlib, seaborn |
